# EduMetrics School Performance Data Pipeline

## Project Description
This project builds an end-to-end data pipeline for analyzing school performance data.

## Tech Stack
- Python
- PySpark
- SQL
- Databricks
- Delta Lake
- GitHub

## Folder Structure
data/raw → raw dataset
sql → SQL queries
notebooks → PySpark scripts
docs → documentation and diagrams
tests → data quality checks
